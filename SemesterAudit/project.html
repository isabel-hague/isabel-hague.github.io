Final Project: City Taste Map

City Taste Map is a modern, editorial-style interactive website that helps users discover restaurants based on vibe rather than traditional reviews. The site uses a minimalist map interface, clean typography, and micro-interactions to create a contemporary dining exploration tool.

Why I'm Making This

  • Combines my interests in design, NYC, food, and psychology
  • Allows me to showcase clean UI + interactive JS functionality
  • Gives me room to use skills I have gained while also expanding my knowledge

Key Features

  • Interactive Map
  • Vibe-based filtering (Romantic, Cozy, Drinks, etc.)
  • Editorial-style restaurant cards
  • Slide-in restaurant detail panel
  • Fully responsive contemporary layout

Site Map

  • Home — intro + vibe filter options
  • Map View — interactive map with restaurant markers
  • Editorial-style restaurant cards
  • Vibe Categories — curated lists (Cozy, Romantic, Trendy, etc.)
  • About the Project — concept, design choices, process
